# Catalogue des datasets - Santé et Territoires Saison 4

## 1. Offre de soins et accessibilité

- **Accessibilité potentielle localisée (APL)**  
  - URL : https://www.data.gouv.fr/datasets/laccessibilite-potentielle-localisee-apl  
  - Type : Offre/demande de soins  
  - Rôle : Indicateur central d'accessibilité aux soins de premier recours, à la commune.

- **Indicateurs Cartosanté (Atlasanté)**  
  - URL : /datasets/67d2eed93872dba3e719d2bd (via SSP Cloud / Atlasanté)  
  - Type : Offre/demande de soins  
  - Rôle : Indicateurs territorialisés (densités, recours, etc.) principalement à la commune.

- **FINESS / Open FINESS / équipements médico-sociaux**  
  - Réexposition des données FINESS : https://www.data.gouv.fr/datasets/reexposition-des-donnees-finess  
  - Open FINESS : https://www.data.gouv.fr/datasets/open-finess  
  - Équipements sociaux et médico-sociaux : https://www.data.gouv.fr/datasets/finess-extraction-des-equipements-sociaux-et-medico-sociaux  
  - Type : Offre de soins et médico-social  
  - Rôle : Localisation et typologie des établissements.

## 2. Déterminants sociaux de la santé

- **Données de population (INSEE)**  
  - URL : https://www.data.gouv.fr/datasets/population  
  - Type : Déterminants sociaux  
  - Rôle : Structure par âge, population, caractéristiques sociodémographiques.

- **Revenus, pauvreté et niveau de vie – données carroyées (FiLoSoFi)**  
  - URL : https://www.data.gouv.fr/datasets/revenus-pauvrete-et-niveau-de-vie-donnees-carroyees  
  - Type : Déterminants sociaux  
  - Rôle : Indicateurs socio-économiques très fins (carreaux 200 m).

- **Revenus et pauvreté des ménages – indicateurs infra-départementaux (FiLoSoFi)**  
  - URL : https://www.data.gouv.fr/datasets/ensemble-de-jeux-de-donnees-relatifs-aux-indicateurs-de-niveau-de-vie-dinegalite-et-de-pauvrete-du-niveau-national-et-infra-departemental  
  - Type : Déterminants sociaux  
  - Rôle : Taux de pauvreté, niveau de vie, inégalités, à des échelons administratifs.

- **Données sur l'emploi et les CSP (statistiques locales INSEE)**  
  - URL : https://statistiques-locales.insee.fr/#c=indicator&i=rp.pt_emplt_cs6&s=2021&t=A01&view=map1  
  - Type : Déterminants sociaux  
  - Rôle : Emploi, chômage, catégories socioprofessionnelles.

- **Indice de défavorisation sociale (FDep) à l'IRIS – Inserm**  
  - URL : https://catalogue.atlasante.fr/geonetwork/srv/fre/catalog.search#/metadata/8ba3a966-fda5-4147-ba45-f6f9fd899c09  
  - Type : Déterminants sociaux  
  - Rôle : Indicateur composite de défavorisation sociale à l'IRIS.

## 3. Mortalité / état de santé

- **Causes médicales de décès CépiDC-INSERM**  
  - URL : https://opendata-cepidc.inserm.fr/  
  - Type : Mortalité  
  - Rôle : Nombre de décès et taux, par cause, territoire, etc.

- **Disparités territoriales de mortalité (INSEE)**  
  - URL : https://www.insee.fr/fr/statistiques/5039901?sommaire=5040030#figure2_radio2  
  - Type : Mortalité  
  - Rôle : Indicateurs d'écarts de mortalité entre territoires.

- **Tables de mortalité par catégorie sociale et par diplôme (INSEE)**  
  - URL : https://www.insee.fr/fr/statistiques/1893092?sommaire=1893101  
  - Type : Mortalité  
  - Rôle : Mise en évidence des inégalités sociales de mortalité.

## 4. Référentiels géographiques

- **Admin Express (IGN)**  
  - URL : https://www.data.gouv.fr/datasets/admin-express-admin-express-cog-admin-express-cog-carto-admin-express-cog-carto-pe-admin-express-cog-carto-plus-pe  
  - Type : Données socles  
  - Rôle : Découpage administratif (communes, EPCI, départements, régions).

- **Contours IRIS (INSEE/IGN)**  
  - URL : https://www.data.gouv.fr/datasets/contours-iris-r-2  
  - Type : Données socles  
  - Rôle : Brique infra-communale pour cartographie fine.

- **Quartiers prioritaires de la politique de la ville (QPV)**  
  - URL : https://www.data.gouv.fr/datasets/quartiers-prioritaires-de-la-politique-de-la-ville-qpv  
  - Type : Données socles  
  - Rôle : Territoires défavorisés ciblés par la politique de la ville.

## 5. Environnement (optionnel)

- **Données du catalogue Green Data For Health**  
  - URL : https://gd4h.ecologie.gouv.fr/catalogue  
  - Type : Données environnementales pour la santé  
  - Rôle : Données environnementales (pollution, etc.) à croiser avec le diagnostic de santé.
